# Imagens SEO - Instruções para Criação

## Imagens Necessárias para SEO Otimizado

### 1. Open Graph Image (og-image.jpg)
- **Dimensões**: 1200x630 pixels
- **Formato**: JPG (alta qualidade)
- **Conteúdo sugerido**:
  - Logo/Nome: "Dr. Dheiver Santos"
  - Subtítulo: "Consultoria em Inteligência Artificial"
  - Background: Gradiente moderno (azul/roxo)
  - Elementos visuais: Ícones de IA, cérebro, circuitos

### 2. Twitter Card Image (twitter-card.jpg)
- **Dimensões**: 1200x600 pixels
- **Formato**: JPG
- **Conteúdo**: Similar ao OG image, otimizada para Twitter

### 3. Logo Principal (logo.svg)
- **Formato**: SVG (vetorial)
- **Conteúdo**: Logo da marca limpo e moderno

### 4. Favicon (favicon.ico)
- **Dimensões**: 32x32 pixels
- **Formato**: ICO

## Como Criar as Imagens

### Opção 1: Ferramentas Online
- **Canva**: Templates profissionais para social media
- **Figma**: Design personalizado
- **Adobe Express**: Templates gratuitos

### Opção 2: Ferramentas de IA
- **Midjourney**: "Professional AI consultant logo, modern tech design"
- **DALL-E**: "Modern technology logo for AI consultant"
- **Stable Diffusion**: Prompts similares

### Opção 3: Serviços de Design
- **Upwork/Fiverr**: Designers freelancers
- **99designs**: Concurso de design

## Implementação Temporária

Enquanto as imagens não são criadas, o sistema usará:
- dheiver-photo.jpg como fallback para OG image
- placeholder.svg para outros casos
- Structured data continuará funcionando normalmente
